Bluetooth Device Scanner

File Description
bluetooth_scan.exe This script is a Bluetooth device scanner that uses Python's Bleak library to scan for nearby Bluetooth devices and extract detailed information about them. It detects and categorizes the devices as either BLE (Bluetooth Low Energy) or Classic Bluetooth by checking the advertised services' UUIDs against a predefined dictionary of common Bluetooth profiles. The script displays information like the device name, MAC address, signal strength (RSSI), connection status, and other relevant details.

Video Format Metadata Sweep

File Description
video-data_scan.exe This Python script uses pymediainfo to extract detailed metadata from video files and scan a folder for media content. The script gathers information about the video and audio tracks, computes file checksums, and formats the output for easy readability. It also allows users to process multiple files in a folder and save metadata to a text file.
