MacOS Unix Bluetooth_scan.exe
Utilizing Python's "Bleak" library to scan for nearby Bluetooth devices and extract detailed information, the Python script identifies between BLE (Bluetooth Low Energy) and Classic Bluetooth platform types by analyzing service UUIDs. For each discovered device, it collects critical details such as the device name, MAC address, RSSI (signal strength), and connection status.

Additionally, the script provides extensive debugging information for developers, such as transmission power, connection interval, security level, and advertised services. This data helps in profiling device connectivity, optimizing performance, and debugging issues during Bluetooth integration or testing.



MacOS Unix Video Format_scan.exe
This Python script uses "pymediainfo" to extract detailed metadata from video files/media content.

The Video Scanner script extracts detailed metadata from video files, including video and audio formats, resolution, bit rate, and file size, while also verifying file integrity with a SHA-256 checksum. It saves the results in a text file and checks device compatibility based on video resolution. It also allows users to process multiple files at once, saving metadata to .txt files.



Spatial-Audio_macos.zip <-> Spatial-Audio_windows.zip
This Unity-based application for macOS and Windows PC lets users immerse themselves in spatial audio. Featuring Head-Related Transfer Function (HRTF) for realistic 3D sound, it adjusts audio based on head orientation, making sound sources feel truly directional.

The demo also incorporates audio occlusion, where objects between the listener and sound sources realistically affect sound perception.
Additionally, it provides distance attenuation, so audio intensity adjusts naturally as users approach or move away from sources. Ideal for experiencing advanced spatial audio in gaming, VR, or simulation environments.
