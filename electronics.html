MacOS Unix Bluetooth_scan.exe - Download

Using Python's "Bleak" library, this script scans for nearby Bluetooth devices and identifies whether they use Bluetooth Low Energy (BLE) or Classic Bluetooth. It gathers key details like device name, MAC address, & signal strength (RSSI). It also collects transmission power and connection status intervals, to help with debugging and improving performance.
MacOS Unix Video Format_scan.exe - Download

This Python script uses "pymediainfo" to gather key details about video files, like their formats, resolution, bit rate, and file size. It also checks file integrity using a SHA-256 checksum. Users can process multiple files at once, and the metadata is saved in .txt files for easy compatibility checks.
